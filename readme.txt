GET HIGH:

Help Goku get high by collecting energy balls generated randomly on the map! The platforms are also randomly generated.

CONTROLS -
A: DASH. Teleport forward from your location by a set amount
UP: JUMP. Can jump twice from the platforms
DOWN: CROUCH. No functionality besides visual
LEFT: RUN LEFT.
RIGHT: RUN RIGHT. 
SELECT: Goes to end game screen
START: Starts the game if on end game screen or main screen
